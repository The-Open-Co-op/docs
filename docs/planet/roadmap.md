# Roadmap

PLANET aims to launch on the First Person Project infrastructure, which is scheduled for initial delivery in September 2026. Between now and then, we have two parallel goals.

## Build the product

We're designing PLANET's UI, onboarding flows, and core apps — and building them as working, coded demos that will plug directly into the First Person Project stack when it's ready.

This means specifying and prototyping the first apps: the vault, contacts and invite flow, chat, First Person Pages, and the Introducer. Each will be a functional demo that shows what PLANET feels like to use, built on a codebase designed to integrate as seamlessly as possible with the PNM, VTA, and trust infrastructure when its ready.

## Build the community

A trust network is only as valuable as the people and communities in it. We're gathering early adopters, contributors, and founding communities now — to be ready for launch.

As the project evolves we will expand the Core Team into collaborative circles within the co-op to work on:

- **Governance** — evolving The Open Co-op's structure as membership grows
- **Product** — designing and building the apps
- **Outreach** — connecting with communities that could benefit from PLANET
- **Network** — recruiting recognisable names and movement leaders to support the project
- **Business** — refining the hosting model, funding strategy, and sustainability plan

If you want to be part of any of these, [join the co-op](https://opencollective.com/the-open-co-op) and get involved.

## Track progress

We track tasks and priorities in [GitHub Projects](https://github.com/The-Open-Co-op/planet/issues).
